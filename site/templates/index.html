Warning!!!

If you stop the generated gcode midprint, restart your printer. It uses a mixture of relative and absolute movements.

Default settings should work with bowden tube.
First try with defaults. Then adjust temps fan etc.

Notes:
  • For delta set bed size to 0 for x and y
  • For direct drive start with .5 starting retraction and .25 incriment
  • For bowden start with .5 starting retraction and .5 incriment
  • To dial in further you can change starting retraction and increments.
